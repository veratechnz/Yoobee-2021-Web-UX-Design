CSS Positioning

ABSOLUTE POSITIONING

I am positioned to the viewport or window.

.absolute {
position: absolute;
width: 200px;
height: 300px;
background-color: #ffff00;
top: 10px;
left: 700px;
padding: 30px;
}

STATIC = DEFAULT POSITIONING

I am the normal div or element position value (in the document).

.normal { /*no positioning value = 'static'*/
width: 500px;
height: 200px;
background-color: #5af7fc;
top: 10px;
right: 200px;
padding: 30px;
}

RELATIVE POSITIONING

I am positioned relative to where I usually am placed.

.relative {
position: relative;
width: 500px;
height: 200px;
background-color: #00ff9d;
top: 80px;
left: 480px;
padding: 30px;
}

INTERNAL ABSOLUTE

FIXED POSITIONING

I am fixed, even if you scroll. I am positioned relative to the window.

.fixed {
position: fixed;
width: 400px;
height: 400px;
padding: 20px;
background-color: #c254d3;
top: 200px;
right: 200px;
color: white;
}

Stacked Elements

Normal - position: static

I will be on the bottom if other positioned elements overlap me.

Absolute - position: absolute

I come in second, I will be on top of non positioned elements!

Relative - position: relative

I am almost the top element, I will overlap static and absolute elements.

Fixed - position: fixed

I am the top element because I am fixed. All other elements will be below me.....unless (scroll down)

Absolute, Relative or Fixed element + z-index: 1 (or more)

I am a relative element with {z-index: 1;}. I will be above all elements on the page including fixed (unless they have a greater assigned z-index number, z-index:2 or more;).
